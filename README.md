# SafeSphere

**Global Open-Source Security Awareness Platform**

SafeSphere is a self-contained platform for phishing simulations, user training, and community engagement.

## Features
- **Phishing Simulations**: Create and run simulated phishing campaigns.
- **User Training**: Interactive training modules with quizzes.
- **Dashboard**: Analytics and reporting for admins and trainers.
- **Gamification**: Badges and leaderboards.

## Tech Stack
- **Backend**: FastAPI (Python)
- **Frontend**: React + TailwindCSS
- **Database**: PostgreSQL
- **Infrastructure**: Docker Compose

## Quick Start

1. Clone the repository
2. Run `docker-compose up --build`
3. Access the application at `http://localhost:5173`
4. Access the API documentation at `http://localhost:8000/docs`
