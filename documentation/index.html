Quick Start

Prerequisites Before you begin, ensure you have Node.js 18+, Python 3.9+, and PostgreSQL 13+ installed.

Installation

Get SafeSphere up and running in minutes:

# Clone the repository
git clone https://github.com/boniyeamincse/SafeSphere.git
cd SafeSphere

# Backend setup
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Frontend setup
cd ../frontend
npm install

# Start the application
npm run dev  # Frontend (port 5173)
cd ../backend && uvicorn app.main:app --reload  # Backend (port 8000)

First Steps

  1. Access the application at http://localhost:5173
  2. Create an admin account
  3. Configure SMTP settings
  4. Create user groups
  5. Launch your first campaign
Tip Check out the User Guide for detailed walkthroughs.
